Role: You are a senior frontend engineer and cinematic web experience architect.

Objective:
Build a production-ready single-page scroll-driven website for Aryan’s Energy, a Pakistani solar technology manufacturer.

Core Mechanic:
The hero section must use a scroll-driven image sequence rendered on HTML canvas.
Frames are located in /public/frames and named frame_0001.webp to frame_TOTAL_FRAMES.webp.
TOTAL_FRAMES = 150

The scroll section must be 500vh tall.
The canvas must be sticky and occupy 100vh during the entire sequence.
Scroll progress (0 to 1) must map precisely to frame index (0 to TOTAL_FRAMES - 1).

Technical Requirements:
- Next.js 14 App Router with TypeScript
- Tailwind CSS
- Framer Motion for scroll tracking
- Canvas rendering with devicePixelRatio scaling for Retina sharpness
- Preload first 20 frames immediately, lazy-load remaining frames in batches
- Use requestAnimationFrame render loop
- Prevent scroll desync
- No video tag usage — must use frame sequence

Design Direction:
Clean, futuristic, industrial.
Dark background (#0b0b0b).
Minimal interface.
High contrast typography.

Typography:
Headings: Orbitron
Body/UI: Rajdhani

Hero Copy:
Headline: Complete Solar Systems. Manufactured in Pakistan.
Subheading: High-efficiency N-Type modules, intelligent inverters, and LiFePO4 storage engineered for South Asia’s toughest climates.
Primary CTA: Request Partnership
Secondary CTA: Download Technical Portfolio

Scroll Phases:

Display headline and subheading.
Minimal layout, left aligned.

75%–100%
Display strong statement:
"Energy Sovereignty. Engineered Locally."
Show CTA buttons clearly.

- Provide hidden text summary for screen readers

Performance:
